<h2 align="center">From Visual Primitives to Semantic Masks: Fine-Grained Visual-Linguistic Alignment for Open-Vocabulary Remote Sensing Image Segmentation</h2>

Official repository for **SANO3**, a training-free framework for fine-grained open-vocabulary semantic segmentation in remote sensing images.

## FG-OVSSRS Benchmark

We introduce **FG-OVSSRS**, a fine-grained open-vocabulary semantic segmentation benchmark for remote sensing images.

The benchmark is publicly available on Hugging Face:

[https://huggingface.co/datasets/blackdyc/FG-OVSSRS](https://huggingface.co/datasets/blackdyc/FG-OVSSRS)

FG-OVSSRS covers diverse remote-sensing scenarios and provides synonym-expanded category queries to evaluate the robustness of OVSS models under flexible real-world open-vocabulary usage.
